### Proyecto backend: API de comercio electrónico

1. Clonar proyecto

2. Ejecutar
```
npm install
```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```

4. Configurar las varibles de entorno

5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar: 
```
npm run start:dev
```

### Stack usado
* Nest JS
* PostgreSQL
